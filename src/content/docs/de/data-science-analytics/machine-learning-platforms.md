# Maschinenlernplattformen

Machine Learning Platforms tools for data science & analytics.

## Verfügbare Werkzeuge

### DataRobot
- **Website**: [https://www.datarobot.com/](https://www.datarobot.com/)
- **Description**: Automatisierte Machine-Learning-Plattform mit KI-agentischen Systemen und Funktionen der Version 10.0.
- **Pricing**: 💰

### H2O.ai
- **Website**: [https://www.h2o.ai/](https://www.h2o.ai/)
- **Description**: Open-Source-ML- und KI-Plattform mit h2oGPT und Document AI.
- **Pricing**: 🆓/💰

### AutoML by Google Cloud
- **Website**: [https://cloud.google.com/automl/](https://cloud.google.com/automl/)
- **Description**: Googles AutoML-Suite mit Vertex AI- und Gemini-Integration.
- **Pricing**: 💰

### Azure ML
- **Website**: [https://azure.microsoft.com/en-us/services/machine-learning/](https://azure.microsoft.com/en-us/services/machine-learning/)
- **Description**: Microsofts ML-Dienst mit Prompt-Flow und Modellkatalog.
- **Pricing**: 💰

### Amazon SageMaker
- **Website**: [https://aws.amazon.com/sagemaker/](https://aws.amazon.com/sagemaker/)
- **Description**: AWS ML-Plattform mit Canvas No-Code-ML und HyperPod.
- **Pricing**: 💰

### Databricks
- **Website**: [https://databricks.com/](https://databricks.com/)
- **Description**: Einheitliche Analyseplattform mit LakehouseIQ und Mosaic AI.
- **Pricing**: 💰

### Dataiku
- **Website**: [https://www.dataiku.com/](https://www.dataiku.com/)
- **Description**: Datenwissenschaftsplattform mit KI-Agenten und LLM-Mesh (2025).
- **Pricing**: 💰

### PyCaret
- **Website**: [https://pycaret.org/](https://pycaret.org/)
- **Description**: Low-Code-ML-Bibliothek für schnelles Experimentieren mit über 30 Algorithmen.
- **Pricing**: 🆓

### Auto-Sklearn 2.0
- **Website**: [https://automl.github.io/auto-sklearn/](https://automl.github.io/auto-sklearn/)
- **Description**: Open-Source AutoML mit Ensemble-Learning.
- **Pricing**: 🆓
