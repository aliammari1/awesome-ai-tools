# Open-Source-LLMs

Open Source LLMs tools for large language models.

## Verfügbare Werkzeuge

### Mistral (Large 3, Ministral, Small)
- **Website**: [https://mistral.ai/](https://mistral.ai/)
- **Description**: Open-Source-Modelle: 675B MoE Groß, kleines multimodales Ministral, Apache 2.0 (2025).
- **Pricing**: 🆓

### DeepSeek (V3.2, R1)
- **Website**: [https://deepseek.com/](https://deepseek.com/)
- **Description**: Open-Source-Reasoning-Modelle, die GPT-5 entsprechen, MoE-Architektur (671B).
- **Pricing**: 🆓

### LLaMA 4 (Scout, Maverick)
- **Website**: [https://ai.meta.com/llama/](https://ai.meta.com/llama/)
- **Description**: Metas MoE-Modelle: 109B Scout (10M Kontext), 400B Maverick multimodal.
- **Pricing**: 🆓

### Qwen 3 (VL-235B)
- **Website**: [https://qwenlm.github.io/](https://qwenlm.github.io/)
- **Description**: Alibabas Billionen-Parameter-Hybrid mit Vision-Sprach-Unterstützung.
- **Pricing**: 🆓

### GLM-4.5
- **Website**: [https://github.com/THUDM/GLM-4](https://github.com/THUDM/GLM-4)
- **Description**: Zhipus 130B Modell, #2 auf LMArena mit MIT-Lizenz.
- **Pricing**: 🆓

### Kimi K2
- **Website**: [https://moonshot.cn/](https://moonshot.cn/)
- **Description**: Moonshot AIs robustes chinesisches LLM mit offener Forschung.
- **Pricing**: 🆓

### Gemma 3
- **Website**: [https://ai.google.dev/gemma](https://ai.google.dev/gemma)
- **Description**: Googles leichtgewichtiges Open-Modell mit Gemma-Lizenz.
- **Pricing**: 🆓

### Falcon 3
- **Website**: [https://falconllm.tii.ae/](https://falconllm.tii.ae/)
- **Description**: Das neueste Open-Source-LLM von TII mit verbesserter Leistung.
- **Pricing**: 🆓
