# مساعدو الترميز بالذكاء الاصطناعي

AI Coding Assistants tools for code generation & development.

## الأدوات المتاحة

### GitHub Copilot
- **Website**: [https://github.com/features/copilot](https://github.com/features/copilot)
- **Description**: مساعد برمجة بالذكاء الاصطناعي مع وكيل الترميز، وضع التخطيط، وتحرير متعدد الملفات (ديسمبر 2025).
- **Pricing**: 💰

### Cursor
- **Website**: [https://cursor.sh/](https://cursor.sh/)
- **Description**: محرر الشيفرة الذي يركز على الذكاء الاصطناعي بقيمة 29.3 مليار دولار مع وضع الصوت وميزات الوكيل وترميز الخلفية (ديسمبر 2025).
- **Pricing**: 🆓/💰

### Windsurf
- **Website**: [https://windsurf.com/](https://windsurf.com/)
- **Description**: مساعد البرمجة بالذكاء الاصطناعي من Codeium مع وكيل Cascade وفهم عميق لقاعدة الشيفرة.
- **Pricing**: 🆓/💰

### Augment Code
- **Website**: [https://augmentcode.com/](https://augmentcode.com/)
- **Description**: منصة ترميز ذكاء اصطناعي للمؤسسات مع سياق يحتوي على أكثر من 10 ملايين رمز ورسوم بيانية للكود.
- **Pricing**: 💰

### Kiro
- **Website**: [https://aws.amazon.com/kiro/](https://aws.amazon.com/kiro/)
- **Description**: بيئة تطوير ذكية من AWS مع تطوير مدفوع بالمواصفات واختبار آلي.
- **Pricing**: 🆓/💰

### Qodo
- **Website**: [https://www.qodo.ai/](https://www.qodo.ai/)
- **Description**: منصة ترميز بالذكاء الاصطناعي لجودة الكود والاختبار والمراجعة (كانت تُعرف سابقًا باسم Codium).
- **Pricing**: 🆓/💰

### Zed
- **Website**: [https://zed.dev/](https://zed.dev/)
- **Description**: محرر كود ذكاء اصطناعي عالي الأداء يتميز بميزات تعاونية ويعتمد على لغة Rust.
- **Pricing**: 🆓

### Tabnine
- **Website**: [https://www.tabnine.com/](https://www.tabnine.com/)
- **Description**: اكتمال كود الذكاء الاصطناعي لأكثر من 80 بيئة تطوير متكاملة مع أمان المؤسسات.
- **Pricing**: 🆓/💰

### Amazon CodeWhisperer
- **Website**: [https://aws.amazon.com/codewhisperer/](https://aws.amazon.com/codewhisperer/)
- **Description**: رفيق ترميز بالذكاء الاصطناعي مع فحص الأمان بواسطة AWS.
- **Pricing**: 🆓/💰

### Jules
- **Website**: [https://labs.google/jules](https://labs.google/jules)
- **Description**: وكيل الترميز بالذكاء الاصطناعي من جوجل للتطوير التلقائي وإصلاح الأخطاء (أكتوبر 2025).
- **Pricing**: 🆓

### Claude Code
- **Website**: [https://anthropic.com/claude-code](https://anthropic.com/claude-code)
- **Description**: أداة الترميز بالذكاء الاصطناعي من Anthropc مع وضع الخطة، تطبيق سطح المكتب، وتدفقات العمل الوكيل (نوفمبر 2025).
- **Pricing**: 💰

### Gemini Code Assist
- **Website**: [https://cloud.google.com/gemini/docs/codeassist](https://cloud.google.com/gemini/docs/codeassist)
- **Description**: الذكاء الاصطناعي للبرمجة من جوجل مع سياق من 2 مليون رمز وميزات للشركات.
- **Pricing**: 💰

### Replit Ghostwriter
- **Website**: [https://replit.com/site/ghostwriter](https://replit.com/site/ghostwriter)
- **Description**: الترميز بالذكاء الاصطناعي في المتصفح مع النشر.
- **Pricing**: 💰

### Codeium
- **Website**: [https://codeium.com/](https://codeium.com/)
- **Description**: تسريع كود الذكاء الاصطناعي مجانًا مع دعم أكثر من 70 بيئة تطوير متكاملة وأكثر من مليون مطور.
- **Pricing**: 🆓

### Continue.dev
- **Website**: [https://continue.dev/](https://continue.dev/)
- **Description**: مساعد برمجة ذكاء اصطناعي مفتوح المصدر لـ VS Code و JetBrains مع نماذج مخصصة.
- **Pricing**: 🆓

### Aider
- **Website**: [https://aider.chat/](https://aider.chat/)
- **Description**: مساعد برمجة ذكاء اصطناعي مفتوح المصدر للترمينال مع تكامل Git.
- **Pricing**: 🆓

### CodeT5
- **Website**: [https://github.com/salesforce/CodeT5](https://github.com/salesforce/CodeT5)
- **Description**: توليد الشيفرة مفتوحة المصدر بواسطة سيلزفورس.
- **Pricing**: 🆓
