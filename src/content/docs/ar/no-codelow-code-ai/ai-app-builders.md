# منشئو تطبيقات الذكاء الاصطناعي

AI App Builders tools for no-code/low-code ai.

## الأدوات المتاحة

### Zapier
- **Website**: [https://zapier.com/](https://zapier.com/)
- **Description**: منصة أتمتة مع وكلاء ذكاء اصطناعي، وروبوتات محادثة، وأكثر من 8000 تكامل للتطبيقات.
- **Pricing**: 🆓/💰

### Make (Integromat)
- **Website**: [https://www.make.com/](https://www.make.com/)
- **Description**: منصة أتمتة بصرية مع سير عمل ذكي و2000+ تطبيق.
- **Pricing**: 🆓/💰

### OutSystems Agent Workbench
- **Website**: [https://www.outsystems.com/](https://www.outsystems.com/)
- **Description**: منصة منخفضة الأكواد لبناء وكلاء الذكاء الاصطناعي المؤسسي (أكتوبر 2025).
- **Pricing**: 💰

### Base44
- **Website**: [https://base44.com/](https://base44.com/)
- **Description**: استحوذت شركة Wix على منشئ تطبيقات الذكاء الاصطناعي مقابل 80 مليون دولار مع تطوير باستخدام اللغة الطبيعية (ديسمبر 2025).
- **Pricing**: 🆓/💰

### Microsoft Power Platform
- **Website**: [https://powerplatform.microsoft.com/](https://powerplatform.microsoft.com/)
- **Description**: التطوير منخفض الكود باستخدام Copilot AI وأكثر من 1000 موصل.
- **Pricing**: 💰

### Google AppSheet
- **Website**: [https://www.appsheet.com/](https://www.appsheet.com/)
- **Description**: تطوير التطبيقات بدون كود باستخدام الذكاء الاصطناعي وتكامل جيميني.
- **Pricing**: 🆓/💰

### Bubble
- **Website**: [https://bubble.io/](https://bubble.io/)
- **Description**: البرمجة البصرية باستخدام ملحقات الذكاء الاصطناعي وقدرات الويب الكاملة.
- **Pricing**: 🆓/💰

### Glide
- **Website**: [https://www.glideapps.com/](https://www.glideapps.com/)
- **Description**: منشئ تطبيقات بدون كود مكون من مكونات الذكاء الاصطناعي وقاعدة بيانات جداول.
- **Pricing**: 🆓/💰

### Adalo
- **Website**: [https://www.adalo.com/](https://www.adalo.com/)
- **Description**: منشئ تطبيقات جوال بدون كود مع ميزات الذكاء الاصطناعي.
- **Pricing**: 🆓/💰

### FlutterFlow
- **Website**: [https://flutterflow.io/](https://flutterflow.io/)
- **Description**: منشئ تطبيقات فلاتر بصري مع توليد التعليمات البرمجية بالذكاء الاصطناعي.
- **Pricing**: 🆓/💰

### Softr
- **Website**: [https://www.softr.io/](https://www.softr.io/)
- **Description**: منشئ تطبيقات ويب بدون كود باستخدام Airtable وذكاء اصطناعي.
- **Pricing**: 🆓/💰
