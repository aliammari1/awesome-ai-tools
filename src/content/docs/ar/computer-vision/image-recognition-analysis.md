# التعرف على الصور وتحليلها

Image Recognition & Analysis tools for computer vision.

## الأدوات المتاحة

### Roboflow
- **Website**: [https://roboflow.com/](https://roboflow.com/)
- **Description**: منصة رؤية حاسوبية مع RF-DETR Seg و AutoML وأكثر من 100 ألف مجموعة بيانات.
- **Pricing**: 🆓/💰

### Google Cloud Vision
- **Website**: [https://cloud.google.com/vision/](https://cloud.google.com/vision/)
- **Description**: واجهة برمجة تطبيقات تحليل الصور مع تكامل Vertex AI و Gemini.
- **Pricing**: 💰

### Clarifai
- **Website**: [https://www.clarifai.com/](https://www.clarifai.com/)
- **Description**: منصة رؤية حاسوبية مع تدريب نماذج مخصصة ودمج نموذج اللغة الكبير.
- **Pricing**: 🆓/💰

### Amazon Rekognition
- **Website**: [https://aws.amazon.com/rekognition/](https://aws.amazon.com/rekognition/)
- **Description**: تحليل الصور باستخدام AWS مع اكتشاف الوجه والنص والمشهد.
- **Pricing**: 💰

### Microsoft Computer Vision
- **Website**: [https://azure.microsoft.com/en-us/services/cognitive-services/computer-vision/](https://azure.microsoft.com/en-us/services/cognitive-services/computer-vision/)
- **Description**: واجهة برمجة تطبيقات الرؤية في أزور مع نموذج فلورنسا الأساسي.
- **Pricing**: 💰

### Averroes.ai
- **Website**: [https://averroes.ai/](https://averroes.ai/)
- **Description**: فحص بصري جاهز للإنتاج للتصنيع ومراقبة الجودة.
- **Pricing**: 💰

### Nanonets
- **Website**: [https://nanonets.com/](https://nanonets.com/)
- **Description**: الذكاء الاصطناعي للمستندات مع التعرف الضوئي القوي على الحروف، ومعالجة الفواتير، والأتمتة.
- **Pricing**: 🆓/💰

### OpenCV
- **Website**: [https://opencv.org/](https://opencv.org/)
- **Description**: مكتبة رؤية حاسوبية مفتوحة المصدر تحتوي على أكثر من 2500 خوارزمية.
- **Pricing**: 🆓

### Viso.ai
- **Website**: [https://viso.ai/](https://viso.ai/)
- **Description**: منصة رؤية حاسوبية بدون كود لتطبيقات المؤسسات.
- **Pricing**: 💰

### Landing AI
- **Website**: [https://landing.ai/](https://landing.ai/)
- **Description**: منصة أندرو نج للرؤية الحاسوبية للتصنيع.
- **Pricing**: 💰
