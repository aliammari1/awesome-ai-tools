# نماذج اللغة الكبيرة مفتوحة المصدر

Open Source LLMs tools for large language models.

## الأدوات المتاحة

### Mistral (Large 3, Ministral, Small)
- **Website**: [https://mistral.ai/](https://mistral.ai/)
- **Description**: النماذج مفتوحة المصدر: 675B MoE كبير، Ministal متعدد الوسائط صغير، Apache 2.0 (2025).
- **Pricing**: 🆓

### DeepSeek (V3.2, R1)
- **Website**: [https://deepseek.com/](https://deepseek.com/)
- **Description**: نماذج الاستدلال مفتوحة المصدر تطابق GPT-5، بنية MoE (671 مليار).
- **Pricing**: 🆓

### LLaMA 4 (Scout, Maverick)
- **Website**: [https://ai.meta.com/llama/](https://ai.meta.com/llama/)
- **Description**: نماذج MoE الخاصة بـ Meta: 109B Scout (سياق 10 مليون)، 400B Maverick متعددة الوسائط.
- **Pricing**: 🆓

### Qwen 3 (VL-235B)
- **Website**: [https://qwenlm.github.io/](https://qwenlm.github.io/)
- **Description**: هجين علي بابا ذو تريليون معامل مع دعم للرؤية واللغة.
- **Pricing**: 🆓

### GLM-4.5
- **Website**: [https://github.com/THUDM/GLM-4](https://github.com/THUDM/GLM-4)
- **Description**: نموذج Zhipu بحجم 130B، المرتبة رقم 2 على LMArena مع رخصة MIT.
- **Pricing**: 🆓

### Kimi K2
- **Website**: [https://moonshot.cn/](https://moonshot.cn/)
- **Description**: نموذج اللغة الكبير الصيني المتين لشركة Moonshot AI مع البحث المفتوح.
- **Pricing**: 🆓

### Gemma 3
- **Website**: [https://ai.google.dev/gemma](https://ai.google.dev/gemma)
- **Description**: النموذج المفتوح خفيف الوزن من جوجل برخصة جيمّا.
- **Pricing**: 🆓

### Falcon 3
- **Website**: [https://falconllm.tii.ae/](https://falconllm.tii.ae/)
- **Description**: أحدث نموذج لغوي مفتوح المصدر من TII مع أداء محسن.
- **Pricing**: 🆓
