# أدوات الذكاء الاصطناعي التعليمية

Educational AI Tools tools for research & education.

## الأدوات المتاحة

### Khan Academy
- **Website**: [https://www.khanacademy.org/](https://www.khanacademy.org/)
- **Description**: منصة تعليمية مع معلم ذكي خانميغو مدعوم بواسطة GPT-4.
- **Pricing**: 🆓

### Coursera
- **Website**: [https://www.coursera.org/](https://www.coursera.org/)
- **Description**: دورات عبر الإنترنت مع توصيات الذكاء الاصطناعي والإرشاد المهني.
- **Pricing**: 🆓/💰

### Duolingo
- **Website**: [https://www.duolingo.com/](https://www.duolingo.com/)
- **Description**: تعلم اللغة باستخدام ميزات الذكاء الاصطناعي Duolingo Max واللعب التمثيلي.
- **Pricing**: 🆓/💰

### Socratic by Google
- **Website**: [https://socratic.org/](https://socratic.org/)
- **Description**: مساعد الواجبات المنزلية بالذكاء الاصطناعي مع شرح خطوة بخطوة.
- **Pricing**: 🆓

### Photomath
- **Website**: [https://photomath.com/](https://photomath.com/)
- **Description**: مُحلل مسائل الرياضيات مع مسح بالكاميرا وتوضيحات.
- **Pricing**: 🆓/💰

### Quizlet
- **Website**: [https://quizlet.com/](https://quizlet.com/)
- **Description**: بطاقات تعليمية وأدوات دراسة مدعومة بالذكاء الاصطناعي مع مدرس Q-Chat.
- **Pricing**: 🆓/💰

### Brainly
- **Website**: [https://brainly.com/](https://brainly.com/)
- **Description**: مساعدة في الواجبات المنزلية مع شروحات الذكاء الاصطناعي وإجابات الخبراء.
- **Pricing**: 🆓/💰

### Synthesis
- **Website**: [https://www.synthesis.com/](https://www.synthesis.com/)
- **Description**: تدريس الرياضيات باستخدام الذكاء الاصطناعي للأطفال مع التركيز على حل المشكلات.
- **Pricing**: 💰

### Cognii
- **Website**: [https://www.cognii.com/](https://www.cognii.com/)
- **Description**: التدريس بالذكاء الاصطناعي مع تقييم اللغة الطبيعية.
- **Pricing**: 💰

### Carnegie Learning
- **Website**: [https://www.carnegielearning.com/](https://www.carnegielearning.com/)
- **Description**: حلول الرياضيات ومحو الأمية المدعومة بالذكاء الاصطناعي للمدارس.
- **Pricing**: 💰
