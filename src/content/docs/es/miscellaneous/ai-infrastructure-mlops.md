# Infraestructura de IA y MLOps

AI Infrastructure & MLOps tools for miscellaneous.

## Herramientas disponibles

### MLflow
- **Website**: [https://mlflow.org/](https://mlflow.org/)
- **Description**: Gestión del ciclo de vida del ML de código abierto con funciones de LLMOps.
- **Pricing**: 🆓

### Kubeflow
- **Website**: [https://www.kubeflow.org/](https://www.kubeflow.org/)
- **Description**: Flujos de trabajo de ML en Kubernetes con pipelines y servicio.
- **Pricing**: 🆓

### DVC
- **Website**: [https://dvc.org/](https://dvc.org/)
- **Description**: Control de versiones de datos para proyectos de ML con seguimiento de experimentos.
- **Pricing**: 🆓

### Weights & Biases
- **Website**: [https://wandb.ai/](https://wandb.ai/)
- **Description**: Seguimiento de experimentos de ML con Weave para la evaluación de LLM.
- **Pricing**: 🆓/💰

### Neptune
- **Website**: [https://neptune.ai/](https://neptune.ai/)
- **Description**: Almacén de metadatos de ML con gestión de experimentos y registro de modelos.
- **Pricing**: 🆓/💰

### Estuary Flow
- **Website**: [https://estuary.dev/](https://estuary.dev/)
- **Description**: Plataforma de automatización de datos en tiempo real con transmisión de baja latencia (2025).
- **Pricing**: 💰

### Informatica
- **Website**: [https://www.informatica.com/](https://www.informatica.com/)
- **Description**: Integración de datos con transformaciones impulsadas por IA y CLAIRE.
- **Pricing**: 💰

### Alteryx
- **Website**: [https://www.alteryx.com/](https://www.alteryx.com/)
- **Description**: Análisis de datos y automatización con el asistente de IA AiDIN.
- **Pricing**: 🆓/💰

### Modal
- **Website**: [https://modal.com/](https://modal.com/)
- **Description**: Nube sin servidor para IA con instancias GPU y escalado automático.
- **Pricing**: 🆓/💰

### Comet ML
- **Website**: [https://www.comet.com/](https://www.comet.com/)
- **Description**: Seguimiento de experimentos de ML con monitoreo de modelos.
- **Pricing**: 🆓/💰

### ClearML
- **Website**: [https://clear.ml/](https://clear.ml/)
- **Description**: Plataforma MLOps de código abierto con seguimiento de experimentos.
- **Pricing**: 🆓/💰

### Seldon
- **Website**: [https://www.seldon.io/](https://www.seldon.io/)
- **Description**: Despliegue y monitoreo de ML a gran escala.
- **Pricing**: 🆓/💰

### BentoML
- **Website**: [https://www.bentoml.com/](https://www.bentoml.com/)
- **Description**: Marco para servir modelos de ML con OpenLLM.
- **Pricing**: 🆓
