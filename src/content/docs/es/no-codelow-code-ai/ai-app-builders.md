# Constructores de aplicaciones de IA

AI App Builders tools for no-code/low-code ai.

## Herramientas disponibles

### Zapier
- **Website**: [https://zapier.com/](https://zapier.com/)
- **Description**: Plataforma de automatización con agentes de IA, chatbots y más de 8000 integraciones de aplicaciones.
- **Pricing**: 🆓/💰

### Make (Integromat)
- **Website**: [https://www.make.com/](https://www.make.com/)
- **Description**: Plataforma de automatización visual con flujos de trabajo de IA y más de 2000 aplicaciones.
- **Pricing**: 🆓/💰

### OutSystems Agent Workbench
- **Website**: [https://www.outsystems.com/](https://www.outsystems.com/)
- **Description**: Plataforma de bajo código para construir agentes de IA empresariales (octubre de 2025).
- **Pricing**: 💰

### Base44
- **Website**: [https://base44.com/](https://base44.com/)
- **Description**: Constructor de aplicaciones de IA adquirido por Wix por 80 millones de dólares con desarrollo en lenguaje natural (dic. 2025).
- **Pricing**: 🆓/💰

### Microsoft Power Platform
- **Website**: [https://powerplatform.microsoft.com/](https://powerplatform.microsoft.com/)
- **Description**: Desarrollo de bajo código con Copilot AI y más de 1000 conectores.
- **Pricing**: 💰

### Google AppSheet
- **Website**: [https://www.appsheet.com/](https://www.appsheet.com/)
- **Description**: Desarrollo de aplicaciones sin código con integración de IA y Gemini.
- **Pricing**: 🆓/💰

### Bubble
- **Website**: [https://bubble.io/](https://bubble.io/)
- **Description**: Programación visual con complementos de IA y capacidades de pila completa.
- **Pricing**: 🆓/💰

### Glide
- **Website**: [https://www.glideapps.com/](https://www.glideapps.com/)
- **Description**: Constructor de aplicaciones sin código con componentes de IA y backend de hojas de cálculo.
- **Pricing**: 🆓/💰

### Adalo
- **Website**: [https://www.adalo.com/](https://www.adalo.com/)
- **Description**: Constructor de aplicaciones móviles sin código con funciones de IA.
- **Pricing**: 🆓/💰

### FlutterFlow
- **Website**: [https://flutterflow.io/](https://flutterflow.io/)
- **Description**: Constructor de aplicaciones Flutter visual con generación de código mediante IA.
- **Pricing**: 🆓/💰

### Softr
- **Website**: [https://www.softr.io/](https://www.softr.io/)
- **Description**: Constructor de aplicaciones web sin código con Airtable e IA.
- **Pricing**: 🆓/💰
