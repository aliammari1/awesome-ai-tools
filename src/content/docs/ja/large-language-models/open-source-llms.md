# オープンソースLLM

Open Source LLMs tools for large language models.

## 利用可能なツール

### Mistral (Large 3, Ministral, Small)
- **Website**: [https://mistral.ai/](https://mistral.ai/)
- **Description**: オープンソースモデル：675B MoE Large、小規模マルチモーダル Ministral、Apache 2.0（2025年）。
- **Pricing**: 🆓

### DeepSeek (V3.2, R1)
- **Website**: [https://deepseek.com/](https://deepseek.com/)
- **Description**: GPT-5に匹敵するオープンソースの推論モデル、MoEアーキテクチャ（671B）。
- **Pricing**: 🆓

### LLaMA 4 (Scout, Maverick)
- **Website**: [https://ai.meta.com/llama/](https://ai.meta.com/llama/)
- **Description**: MetaのMoEモデル：109B Scout（10Mコンテキスト）、400B Maverickマルチモーダル。
- **Pricing**: 🆓

### Qwen 3 (VL-235B)
- **Website**: [https://qwenlm.github.io/](https://qwenlm.github.io/)
- **Description**: Alibabaの兆パラメータを持つ視覚-言語対応ハイブリッド。
- **Pricing**: 🆓

### GLM-4.5
- **Website**: [https://github.com/THUDM/GLM-4](https://github.com/THUDM/GLM-4)
- **Description**: Zhipuの130Bモデル、MITライセンスでLMArenaで2位。
- **Pricing**: 🆓

### Kimi K2
- **Website**: [https://moonshot.cn/](https://moonshot.cn/)
- **Description**: Moonshot AIのオープンリサーチによる強力な中国語大規模言語モデル。
- **Pricing**: 🆓

### Gemma 3
- **Website**: [https://ai.google.dev/gemma](https://ai.google.dev/gemma)
- **Description**: Gemmaライセンス付きのGoogleの軽量オープンモデル。
- **Pricing**: 🆓

### Falcon 3
- **Website**: [https://falconllm.tii.ae/](https://falconllm.tii.ae/)
- **Description**: パフォーマンスが向上したTIIによる最新のオープンソースLLM。
- **Pricing**: 🆓
