# Assistants de codage IA

AI Coding Assistants tools for code generation & development.

## Outils disponibles

### GitHub Copilot
- **Website**: [https://github.com/features/copilot](https://github.com/features/copilot)
- **Description**: Programmeur en binôme IA avec agent de codage, mode plan et édition multi-fichiers (décembre 2025).
- **Pricing**: 💰

### Cursor
- **Website**: [https://cursor.sh/](https://cursor.sh/)
- **Description**: Éditeur de code axé sur l'IA évalué à 29,3 milliards de dollars avec mode vocal, fonctionnalités d'agent et codage en arrière-plan (décembre 2025).
- **Pricing**: 🆓/💰

### Windsurf
- **Website**: [https://windsurf.com/](https://windsurf.com/)
- **Description**: L'assistant de codage IA de Codeium avec agent Cascade et compréhension approfondie de la base de code.
- **Pricing**: 🆓/💰

### Augment Code
- **Website**: [https://augmentcode.com/](https://augmentcode.com/)
- **Description**: Plateforme de codage IA pour les entreprises avec un contexte de plus de 10 millions de jetons et un graphe de code.
- **Pricing**: 💰

### Kiro
- **Website**: [https://aws.amazon.com/kiro/](https://aws.amazon.com/kiro/)

- **Pricing**: 🆓/💰

### Qodo
