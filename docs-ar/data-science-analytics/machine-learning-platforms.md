# منصات التعلم الآلي

Machine Learning Platforms tools for data science & analytics.

## الأدوات المتاحة

### DataRobot
- **Website**: [https://www.datarobot.com/](https://www.datarobot.com/)
- **Description**: منصة التعلم الآلي المؤتمتة مع أنظمة الذكاء الاصطناعي الوكيلة وميزات الإصدار 10.0.
- **Pricing**: 💰

### H2O.ai
- **Website**: [https://www.h2o.ai/](https://www.h2o.ai/)
- **Description**: منصة مفتوحة المصدر للتعلم الآلي والذكاء الاصطناعي مع h2oGPT و Document AI.
- **Pricing**: 🆓/💰

### AutoML by Google Cloud
- **Website**: [https://cloud.google.com/automl/](https://cloud.google.com/automl/)
- **Description**: مجموعة AutoML من جوجل مع تكامل Vertex AI و Gemini.
- **Pricing**: 💰

### Azure ML
- **Website**: [https://azure.microsoft.com/en-us/services/machine-learning/](https://azure.microsoft.com/en-us/services/machine-learning/)
- **Description**: خدمة التعلم الآلي من مايكروسوفت مع تدفق المطالبات وفهرس النماذج.
- **Pricing**: 💰

### Amazon SageMaker
- **Website**: [https://aws.amazon.com/sagemaker/](https://aws.amazon.com/sagemaker/)
- **Description**: منصة AWS للتعلم الآلي مع Canvas للتعلم الآلي بدون كود و HyperPod.
- **Pricing**: 💰

### Databricks
- **Website**: [https://databricks.com/](https://databricks.com/)
- **Description**: منصة تحليلات موحدة مع LakehouseIQ وMosaic AI.
- **Pricing**: 💰

### Dataiku
- **Website**: [https://www.dataiku.com/](https://www.dataiku.com/)
- **Description**: منصة علوم البيانات مع وكلاء الذكاء الاصطناعي وشبكة LLM (2025).
- **Pricing**: 💰

### PyCaret
- **Website**: [https://pycaret.org/](https://pycaret.org/)
- **Description**: مكتبة تعلم آلي منخفضة الشيفرة للتجارب السريعة مع أكثر من 30 خوارزمية.
- **Pricing**: 🆓

### Auto-Sklearn 2.0
- **Website**: [https://automl.github.io/auto-sklearn/](https://automl.github.io/auto-sklearn/)
- **Description**: التعلم الآلي التلقائي مفتوح المصدر مع التعلم الجماعي.
- **Pricing**: 🆓
