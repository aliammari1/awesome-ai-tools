# إدارة المهام والمشاريع

Task & Project Management tools for productivity.

### Notion

- **Description**: مساحة عمل شاملة مع وكلاء الذكاء الاصطناعي والأسئلة والأجوبة (نوشن 3.0).
- **Pricing**: 🆓/💰

- **Description**: جدولة المهام بلغة طبيعية باستخدام الذكاء الاصطناعي.
- **Pricing**: 🆓/💰
