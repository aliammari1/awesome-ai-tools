# Constructeurs d'applications d'IA

AI App Builders tools for no-code/low-code ai.

## Outils disponibles

### Zapier
- **Website**: [https://zapier.com/](https://zapier.com/)
- **Description**: Plateforme d'automatisation avec des agents IA, des chatbots et plus de 8000 intégrations d'applications.
- **Pricing**: 🆓/💰

### Make (Integromat)
- **Website**: [https://www.make.com/](https://www.make.com/)
- **Description**: Plateforme d'automatisation visuelle avec des flux de travail IA et plus de 2000 applications.
- **Pricing**: 🆓/💰

### OutSystems Agent Workbench
- **Website**: [https://www.outsystems.com/](https://www.outsystems.com/)
- **Description**: Plateforme low-code pour créer des agents IA d'entreprise (octobre 2025).
- **Pricing**: 💰

### Base44
- **Website**: [https://base44.com/](https://base44.com/)
- **Description**: Constructeur d'applications IA acquis par Wix pour 80 millions de dollars avec développement en langage naturel (décembre 2025).
- **Pricing**: 🆓/💰

### Microsoft Power Platform
- **Website**: [https://powerplatform.microsoft.com/](https://powerplatform.microsoft.com/)

- **Pricing**: 💰
