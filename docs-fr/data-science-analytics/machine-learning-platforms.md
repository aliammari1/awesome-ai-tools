# Plateformes d'apprentissage automatique

Machine Learning Platforms tools for data science & analytics.

## Outils disponibles

### DataRobot
- **Website**: [https://www.datarobot.com/](https://www.datarobot.com/)
- **Description**: Plateforme d'apprentissage automatique automatisée avec des systèmes agentiques d'IA et des fonctionnalités de la version 10.0.
- **Pricing**: 💰

### H2O.ai
- **Website**: [https://www.h2o.ai/](https://www.h2o.ai/)
- **Description**: Plateforme ML et IA open-source avec h2oGPT et Document AI.
- **Pricing**: 🆓/💰

### AutoML by Google Cloud
- **Website**: [https://cloud.google.com/automl/](https://cloud.google.com/automl/)
- **Description**: La suite AutoML de Google avec intégration de Vertex AI et Gemini.
- **Pricing**: 💰

### Azure ML
- **Website**: [https://azure.microsoft.com/en-us/services/machine-learning/](https://azure.microsoft.com/en-us/services/machine-learning/)
- **Description**: Le service ML de Microsoft avec flux d'invite et catalogue de modèles.
- **Pricing**: 💰

### Amazon SageMaker
- **Website**: [https://aws.amazon.com/sagemaker/](https://aws.amazon.com/sagemaker/)
- **Description**: Plateforme ML d'AWS avec Canvas ML sans code et HyperPod.
- **Pricing**: 💰

### Databricks
- **Website**: [https://databricks.com/](https://databricks.com/)
- **Description**: Plateforme d'analyse unifiée avec LakehouseIQ et Mosaic AI.
- **Pricing**: 💰

### Dataiku
- **Website**: [https://www.dataiku.com/](https://www.dataiku.com/)
- **Description**: Plateforme de science des données avec agents IA et réseau LLM (2025).
- **Pricing**: 💰

### PyCaret
- **Website**: [https://pycaret.org/](https://pycaret.org/)
- **Description**: Bibliothèque ML low-code pour des expérimentations rapides avec plus de 30 algorithmes.
- **Pricing**: 🆓

### Auto-Sklearn 2.0
- **Website**: [https://automl.github.io/auto-sklearn/](https://automl.github.io/auto-sklearn/)
- **Description**: AutoML open-source avec apprentissage par ensemble.
- **Pricing**: 🆓
