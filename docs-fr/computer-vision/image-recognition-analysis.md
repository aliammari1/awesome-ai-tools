# Reconnaissance et analyse d'images

Image Recognition & Analysis tools for computer vision.

## Outils disponibles

### Roboflow
- **Website**: [https://roboflow.com/](https://roboflow.com/)
- **Description**: Plateforme de vision par ordinateur avec RF-DETR Seg, AutoML et plus de 100 000 ensembles de données.
- **Pricing**: 🆓/💰

### Google Cloud Vision
- **Website**: [https://cloud.google.com/vision/](https://cloud.google.com/vision/)
- **Description**: API d'analyse d'images avec intégration Vertex AI et Gemini.
- **Pricing**: 💰

### Clarifai
- **Website**: [https://www.clarifai.com/](https://www.clarifai.com/)
- **Description**: Plateforme de vision par ordinateur avec entraînement de modèles personnalisés et intégration de LLM.
- **Pricing**: 🆓/💰

### Amazon Rekognition
- **Website**: [https://aws.amazon.com/rekognition/](https://aws.amazon.com/rekognition/)
- **Description**: Analyse d'images AWS avec détection de visages, de textes et de scènes.
- **Pricing**: 💰

### Microsoft Computer Vision
- **Website**: [https://azure.microsoft.com/en-us/services/cognitive-services/computer-vision/](https://azure.microsoft.com/en-us/services/cognitive-services/computer-vision/)
- **Description**: API de vision Azure avec le modèle de fondation Florence.
- **Pricing**: 💰

### Averroes.ai
- **Website**: [https://averroes.ai/](https://averroes.ai/)
- **Description**: Inspection visuelle prête pour la production pour la fabrication et le contrôle qualité.
- **Pricing**: 💰

### Nanonets
- **Website**: [https://nanonets.com/](https://nanonets.com/)
- **Description**: IA documentaire avec OCR puissant, traitement des factures et automatisation.
- **Pricing**: 🆓/💰

### OpenCV
- **Website**: [https://opencv.org/](https://opencv.org/)
- **Description**: Bibliothèque de vision par ordinateur open-source avec plus de 2500 algorithmes.
- **Pricing**: 🆓

### Viso.ai
- **Website**: [https://viso.ai/](https://viso.ai/)
- **Description**: Plateforme de vision par ordinateur sans code pour les applications d'entreprise.
- **Pricing**: 💰

### Landing AI
- **Website**: [https://landing.ai/](https://landing.ai/)
- **Description**: La plateforme de vision par ordinateur d'Andrew Ng pour la fabrication.
- **Pricing**: 💰
