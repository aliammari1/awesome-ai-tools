# 人工智能基础设施与机器学习运维

AI Infrastructure & MLOps tools for miscellaneous.

## 可用工具

### MLflow
- **Website**: [https://mlflow.org/](https://mlflow.org/)
- **Description**: 具有 LLMOps 功能的开源机器学习生命周期管理。
- **Pricing**: 🆓

### Kubeflow
- **Website**: [https://www.kubeflow.org/](https://www.kubeflow.org/)
- **Description**: 在 Kubernetes 上使用管道和服务的机器学习工作流。
- **Pricing**: 🆓

### DVC
- **Website**: [https://dvc.org/](https://dvc.org/)
- **Description**: 用于机器学习项目的实验跟踪的数据版本控制。
- **Pricing**: 🆓

### Weights & Biases
- **Website**: [https://wandb.ai/](https://wandb.ai/)
- **Description**: 使用 Weave 进行 LLM 评估的机器学习实验跟踪。
- **Pricing**: 🆓/💰

### Neptune
- **Website**: [https://neptune.ai/](https://neptune.ai/)
- **Description**: 具有实验管理和模型注册功能的机器学习元数据存储。
- **Pricing**: 🆓/💰

### Estuary Flow
- **Website**: [https://estuary.dev/](https://estuary.dev/)
- **Description**: 具有低延迟流处理的实时数据自动化平台（2025年）。
- **Pricing**: 💰

### Informatica
- **Website**: [https://www.informatica.com/](https://www.informatica.com/)
- **Description**: 使用人工智能驱动的转换和CLAIRE进行数据整合。
- **Pricing**: 💰

### Alteryx
- **Website**: [https://www.alteryx.com/](https://www.alteryx.com/)
- **Description**: 使用 AI 助手 AiDIN 进行数据分析和自动化。
- **Pricing**: 🆓/💰

### Modal
- **Website**: [https://modal.com/](https://modal.com/)
- **Description**: 用于人工智能的无服务器云，提供 GPU 实例和自动扩展功能。
- **Pricing**: 🆓/💰

### Comet ML
- **Website**: [https://www.comet.com/](https://www.comet.com/)
- **Description**: 机器学习实验跟踪与模型监控。
- **Pricing**: 🆓/💰

### ClearML

### Seldon
