# Reconocimiento y análisis de imágenes

Image Recognition & Analysis tools for computer vision.

## Herramientas disponibles

### Roboflow
- **Website**: [https://roboflow.com/](https://roboflow.com/)
- **Description**: Plataforma de visión por computadora con RF-DETR Seg, AutoML y más de 100.000 conjuntos de datos.
- **Pricing**: 🆓/💰

### Google Cloud Vision
- **Website**: [https://cloud.google.com/vision/](https://cloud.google.com/vision/)
- **Description**: API de análisis de imágenes con integración de Vertex AI y Gemini.
- **Pricing**: 💰

### Clarifai
- **Website**: [https://www.clarifai.com/](https://www.clarifai.com/)
- **Description**: Plataforma de visión por computadora con entrenamiento de modelos personalizados e integración de LLM.
- **Pricing**: 🆓/💰

### Amazon Rekognition
- **Website**: [https://aws.amazon.com/rekognition/](https://aws.amazon.com/rekognition/)
- **Description**: Análisis de imágenes de AWS con detección de rostro, texto y escena.
- **Pricing**: 💰

### Microsoft Computer Vision
- **Website**: [https://azure.microsoft.com/en-us/services/cognitive-services/computer-vision/](https://azure.microsoft.com/en-us/services/cognitive-services/computer-vision/)
- **Description**: API de visión de Azure con el modelo base Florence.
- **Pricing**: 💰

### Averroes.ai
- **Website**: [https://averroes.ai/](https://averroes.ai/)
- **Description**: Inspección visual lista para producción para fabricación y control de calidad.
- **Pricing**: 💰

### Nanonets
- **Website**: [https://nanonets.com/](https://nanonets.com/)
- **Description**: IA de documentos con OCR potente, procesamiento de facturas y automatización.
- **Pricing**: 🆓/💰

### OpenCV
- **Website**: [https://opencv.org/](https://opencv.org/)
- **Description**: Biblioteca de visión por computadora de código abierto con más de 2500 algoritmos.
- **Pricing**: 🆓

### Viso.ai
- **Website**: [https://viso.ai/](https://viso.ai/)
- **Description**: Plataforma de visión por computadora sin código para aplicaciones empresariales.
- **Pricing**: 💰

### Landing AI
- **Website**: [https://landing.ai/](https://landing.ai/)
- **Description**: La plataforma de visión por computadora de Andrew Ng para la fabricación.
- **Pricing**: 💰
