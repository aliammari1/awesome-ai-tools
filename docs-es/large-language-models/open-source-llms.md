# LLM de código abierto

Open Source LLMs tools for large language models.

## Herramientas disponibles

### Mistral (Large 3, Ministral, Small)
- **Website**: [https://mistral.ai/](https://mistral.ai/)
- **Description**: Modelos de código abierto: 675B MoE Large, pequeño multimodal Ministral, Apache 2.0 (2025).
- **Pricing**: 🆓

### DeepSeek (V3.2, R1)
- **Website**: [https://deepseek.com/](https://deepseek.com/)
- **Description**: Modelos de razonamiento de código abierto que igualan a GPT-5, arquitectura MoE (671B).
- **Pricing**: 🆓

### LLaMA 4 (Scout, Maverick)
- **Website**: [https://ai.meta.com/llama/](https://ai.meta.com/llama/)
- **Description**: Modelos MoE de Meta: 109B Scout (10M ctx), 400B Maverick multimodal.
- **Pricing**: 🆓

### Qwen 3 (VL-235B)
