# Plataformas de Aprendizaje Automático

Machine Learning Platforms tools for data science & analytics.

## Herramientas disponibles

### DataRobot
- **Website**: [https://www.datarobot.com/](https://www.datarobot.com/)
- **Description**: Plataforma de aprendizaje automático automatizado con sistemas agentivos de IA y características de la versión 10.0.
- **Pricing**: 💰

### H2O.ai
- **Website**: [https://www.h2o.ai/](https://www.h2o.ai/)
- **Description**: Plataforma de ML e IA de código abierto con h2oGPT y Document AI.
- **Pricing**: 🆓/💰

### AutoML by Google Cloud
- **Website**: [https://cloud.google.com/automl/](https://cloud.google.com/automl/)
- **Description**: La suite AutoML de Google con integración de Vertex AI y Gemini.
- **Pricing**: 💰

### Azure ML
- **Website**: [https://azure.microsoft.com/en-us/services/machine-learning/](https://azure.microsoft.com/en-us/services/machine-learning/)
- **Description**: El servicio de ML de Microsoft con flujo de indicaciones y catálogo de modelos.
- **Pricing**: 💰

### Amazon SageMaker
- **Website**: [https://aws.amazon.com/sagemaker/](https://aws.amazon.com/sagemaker/)
- **Description**: Plataforma de ML de AWS con Canvas de ML sin código y HyperPod.
- **Pricing**: 💰

### Databricks
- **Website**: [https://databricks.com/](https://databricks.com/)
- **Description**: Plataforma de análisis unificada con LakehouseIQ y Mosaic AI.
- **Pricing**: 💰

### Dataiku
- **Website**: [https://www.dataiku.com/](https://www.dataiku.com/)
- **Description**: Plataforma de ciencia de datos con agentes de IA y Malla LLM (2025).
- **Pricing**: 💰

### PyCaret
- **Website**: [https://pycaret.org/](https://pycaret.org/)
- **Description**: Biblioteca de ML de bajo código para experimentación rápida con más de 30 algoritmos.
- **Pricing**: 🆓

### Auto-Sklearn 2.0
- **Website**: [https://automl.github.io/auto-sklearn/](https://automl.github.io/auto-sklearn/)
- **Description**: AutoML de código abierto con aprendizaje por conjunto.
- **Pricing**: 🆓
